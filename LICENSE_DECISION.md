# License decision

No license was automatically assigned because licensing is a legal and
authorship decision.

A common open-science arrangement is:

- a software license for original code, such as MIT or BSD-3-Clause;
- a data license for author-owned datasets, such as CC BY 4.0;
- the original license for third-party code and data.

Verify institutional, co-author, event, and third-party requirements before
publishing.
